## Steps on How to Create a GitHub Page
1. Create new Repository in GitHub.
2. Enter <username>.github.io as the repository name (e.g int-gcr.github.io).
3. Choose repository visbility (either Private or Public) and toggle README on.
4. Then create repository.
5. Go to Setting and in the "Code and Automation" section click Pages.
6. In "Build and Deployment" and under Source, pick Deploy from branch.
7. Under branch select a publishing source (e.g main).
